# SearcherCompanyByNIP
GUI (Java fx) client based on WebService GUS (https://wyszukiwarkaregon.stat.gov.pl).
 

Returns following info about company you found
-

* fullName,
* shortenName,
* country,
* state,
* city,
* postcode,
* street,
* street number,
* house number,
* phone,
* e-mail,
* creation date,
* end date.
